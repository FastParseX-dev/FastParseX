FastParseX Architecture

FastParseX is organized into modular components to ensure clarity, performance, and maintainability.

Modules:
- core — shared utilities and base components
- csv — CSV parsing engine
- log — log file parser
- binary — binary data parser
- compress — compression support (gzip, xz, zstd)
- parallel — parallel execution engine
- profiling — performance metrics and statistics
- export — Arrow and Parquet export

Each module is isolated and can be extended independently.
