FastParseX Examples
This directory contains usage examples demonstrating how to work with the FastParseX parsing engine.

Available examples:

- example_csv.cpp — basic CSV parsing

- example_parallel_csv.cpp — multi‑threaded CSV parsing (when supported)

- example_log.cpp — log file parsing

- example_binary.cpp — binary file reading

- example_parallel_log.cpp — parallel log parsing

- example_export.cpp — basic export functionality (Community Edition)

More examples will be added in future releases.
